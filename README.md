# Airport Simulation 🛫

A logistics tycoon game inspired by the massive scale and complexity of **Chicago O'Hare International Airport**. Build your terminals, manage complex traffic flows, and solve operational crises in real-time.

## 🚀 Features
- **Custom Logistics:** Pave your own private expressways to bypass city gridlock.
- **Branding:** Customize your own highway exit gantries (e.g., "Exit 777").
- **Crisis Management:** Handle winter blizzards and traffic pileups with AI maintenance crews or manual 3-second interactive intervention holds!
- **Tactical Construction:** Real-time building (4-5 mins) and road paving (3 mins) timers to challenge your long-term planning.

## 🛠️ Tech Stack
- **iOS/macOS:** Swift, SwiftUI, GameplayKit.
- **Windows:** C++ with SFML.
- **Build System:** XcodeGen (`project.yml`).
